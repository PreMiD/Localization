# সিস্টেম রিকোয়ারমেন্টগুলো

> এইসব তথ্যগুলো শুধুমাত্র অ্যাপ্লিকেশান এর সাথে সম্পর্কিত। 
> 
> {.is-info}

PreMiD কার্যত যেকোনো সিস্টেমেই চলে। এর মানে হচ্ছে এটা **লিনাক্স**, **ম্যাকওএস**, **উইন্ডোজ** এর পাশাপাশি কিছু **লিনাক্স পরিবর্তনসমূহতে** চলে।

### প্রসেসর
PreMiD পুরোপুরি ঠিক চলে একটি প্রসেসর কোরে।

### র‍্যাম
লিনাক্স সিস্টেমগুলোর **কমপক্ষে 20 এমবি র‍্যাম** থাকা উচিত PreMiD চালানোর জন্যে। উইন্ডোজ এবং ম্যাকওএস সিস্টেমগুলোর সাধারণত একটু বেশি র‍্যাম এর প্রয়োজন হয়।

### স্টোরেজ
PreMiD **150 এমবি** হার্ড ডিস্ক স্টোরেজ নেয়। যাহোক কমপক্ষে PreMiD - এর জন্য **200 এমবি** স্টোরেজ পরামর্শিত।

### ইন্টারনেট অ্যাক্সেস
একটি ইন্টারনেট কানেকশন প্রয়োজন PreMiD চালানোর জন্যে। PreMiD অটোমেটিকভাবে চেক করবে নতুন আপডেট এর জন্য, ভাষা, Presence আপডেট ইত্যাদি মাঝে মাঝে।
