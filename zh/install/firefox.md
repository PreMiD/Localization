安装扩展程序非常重要，因为应用程序本身无法做任何事情。

# 安装
1. [点击这里](https://premid.app/downloads) 来转到我们的下载页面。
2. 选择 **Firefox**。
3. 搜索 `关于:addons`。
4. 点击右上角的小装备图标。
5. 选择 **从文件安装** 附加组件.
6. 选择您最近下载的文件。
7. 点击 **添加** 来接受安装。

> 不要忘记 [安装 **应用程序**](/install)。
